# Markdown Lists

## Unordered Lists

- 📝 Item 1
- 📚 Item 2
  - 🔹 Subitem 1
  - 🔹 Subitem 2

* 🌟 Item 1
* 🌟 Item 2
  * 🔸 Subitem 1
  * 🔸 Subitem 2
    

+ 🎯 Item 1
+ 🎯 Item 2
  + 🔹 Subitem 1
  + 🔹 Subitem 2

## Ordered Lists

1. 🥇 First item
2. 🥈 Second item
   1. 🔹 Subitem 1
   2. 🔹 Subitem 2
3. 🥉 Third item

## Definition Lists

**Term 1**  
🔍 Definition 1

**Term 2**  
🔍 Definition 2
  * ✨ Subdefinition 1
  * ✨ Subdefinition 2

## Checklists

- [ ] 🛠️ Task 1
- [x] ✅ Task 2
  - [ ] 🔧 Subtask 1
  - [x] ✅ Subtask 2

## Nested Lists

1. 📋 First item
   - 🔸 Subitem 1
   - 🔸 Subitem 2
     1. 🔹 Sub-subitem 1
     2. 🔹 Sub-subitem 2
2. 📂 Second item
   * 🔹 Subitem 1
   * 🔹 Subitem 2

## Combining Lists

- 🌐 Item 1
- 🌐 Item 2
  1. 🔹 Subitem 1
  2. 🔹 Subitem 2
- 🌟 Item 3
